# Newsletter Buddy

Aplicación full-stack para gestión de newsletters.

## Tecnologías

### Backend
- Node.js + Express
- TypeScript
- MongoDB

### Frontend
- React + Vite
- TypeScript
- Styled Components

## Instalación

```bash
# Con Docker
docker-compose -f docker-compose.local.yml up

# Manual
cd NewsletterBuddyBackend && npm install && npm run dev
cd NewsletterBuddyFrontend && npm install && npm run dev
```

## Funcionalidades

- Agregar suscriptores
- Enviar newsletters
- Analytics de emails
- Página de cancelación de suscripción

## Propósito

Proyecto full-stack para practicar desarrollo con TypeScript.
